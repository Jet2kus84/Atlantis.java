Summary

AtlantisEngine is an experimental game engine written in many languages with the same API. The goal of this project is to make an easy and simple game engine. The project is composed by two main modules.

The Framework

It's close to XNA/MonoGame Framework API without some features like network, advanced 3D, gamers services, etc.

The Engine

The engine is a modified port of YnaEngine, another Game Engine that I written. The API of the engine is similar to Flixel, an awesome ActionScript 3 game framework. You have a global static class with many usefull objects.

Some features

  • State management
  • Sprite and animated Sprite
  • Input management (keyboard, mouse)
